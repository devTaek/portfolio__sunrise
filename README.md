# SUNRISE SOCCER TEAM MANAGE SERVICE
>"Sunrise" 프로젝트는 React와 Express를 사용한 웹 애플리케이션입니다.

## ✅ 프로젝트 소개
>"풋살 동호회 회원 관리 및 회계 서비스" 를 주제로 제작한 리액트 웹 서비스입니다.
팀 회원들의 정보 및 투명한 회비 관리를 목적으로 제작했습니다.
